About Me

Hi! I'm Maxine Audrey Pulao.

I am currently a third year college student at Technological Institute of the Philippines - Quezon City. My Elective track is System Administration and at the present I have just finished the course CPE232 - Managing Eneterprises and Servers. This Portfolio contains all my hard work, which are my Hands On Activities. In which I am proudly compiling in this website.

Preliminary

Students will be understanding more on DevOps culture. They will be also working with Git and Github and leveraging Infrastructure as Code with Ansible

Hands On Activity 0

This activity is allocated for students to create and configure Virtual Machines in Microsoft Azure or VirtualBox. This is also an introduction on setting-up a Virtual Network and Test the Connectivity of VMs

Hands On Activity 2

This activity is for students to configure remote and local machine to connect via SSH using a KEY instead of using a password, create a public key and private key and lastly, verify its connectivity. Aside from that, this is to setup their Git Repository using their local and remote repositories and configure and Run ad hoc commands from local machine to remote servers

Hands On Activity 3

This Activity is for students to innstall Community Enterprise OS or Red Hat Linux and configure remote SSH connection from remote computer to CentOS/RHEL-8

Hands On Activity 4

This Activity is for students to use commands that makes changes to remote machines and use playbook in automating ansible commands

Hands On Activity 5

This Activity is for students to use "when" command in playbook for different OS distributions and then apply refactoring techniques in cleaning up the playbook codes

PRELIM EXAM

Midterm

Students will be installing, configuring and manage MIS services, common workload services, availability monitoring, performance monitoring and log monitoring

Hands On Activity 6

Students will learn how to individualize hosts, apply tags in selecting plays to run and managing Services from remote servers using playbooks

Hands On Activity 7

Students will learn how to manage files in remote servers and implement roles in ansible

Hands On Activity 8

Students will learn how to Create and design a workflow that installs, configure and manage enterprise monitoring tools using Ansible as an Infrastructure as Code (IaC) tool. Installing Nagios.

Hands On Activity 9

Students will learn how to create and design a workflow that installs, configure and manage enterprise monitoring tools using Ansible as an Infrastructure as Code (IaC) tool. Installing Prometheus

Hands On Activity 10

Students will learn how to create and design a workflow that installs, configure and manage enterprise monitoring tools using Ansible as an Infrastructure as Code (IaC) tool. Installing Elastic Stack in Seperate Hosts (Elastic Search, Kibana, Logstash)

MIDTERM EXAM

Finals

Students will be able to configure and manage containerized environment and an introduction to cloud Services

Hands On Activity 11

Students will learn how to create a Dockerfile and form a workflow using Ansible as Infrastructure as Code (IaC) to enable Continuous Delivery process

Hands On Activity 12

Students will learn how to review basic bash scripting techniques because bash scripting is a prerequisite for the rest of the lab. They will then build and modify a Python script for a simple web application. Next, they will create a bash script to automate the process for creating a Dockerfile, building the Docker container, and running the Docker container. Finally, they will use docker commands to investigate the intricacies of the Docker container instance.

Hands On Activity 13

Students will learn how to create a workflow to install OpenStack using Ansible as your Infrastructure as Code (IaC). Creating a playbook that conatains NTP, Openstack Packages, SQL Database, Message Queue and Memcached

Hands On Activity 14

Students will learn how to create a workflow to install OpenStack using Ansible as your Infrastructure as Code (IaC). Creating a playbook that conatains Keystone, Glance and Nova

Hands On Activity 15

Students will learn how to create a workflow to install OpenStack using Ansible as your Infrastructure as Code (IaC). Creating a playbook that conatains Neutron, Horizon and Cinder

FINALS EXAM

My Thoughts On This Subject

Back on August 2022, I was nervous on taking this subject since it was my first face-to-face class since the pandemic happened. Aside from that, I was not really confident since the online set-up does not work for me sometimes and I was not confident on my knowledge on System Administration 1. The past few months had many tedious Hands-On Activities which require the students's full attention on completing the activity. Even though I have encountered many difficulties and struggles to finish some of my activities, I still managed to learn and little by little complete them. This subject taught me a lot in Managing Enterprises and Servers but not only that, it also taught me to gather a lot of patience and determination which is needed to face my future. But the most important thing that I have gained in this subject is the recovery to study and do my best again which I lost over the last two years of online classes. I would like to thank my Professor, Dr. Jonathan Taylar for being a patient with me and considerate in times of need. Lastly, I would like to thank my classmates for always helping me. I owe them this subject. Thank you.
